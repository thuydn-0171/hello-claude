# API Designer Agent

A custom agent for designing and documenting REST API endpoints.

## Purpose
Helps design new API endpoints following project standards.

## Capabilities
- Design RESTful endpoints
- Generate OpenAPI/Swagger documentation
- Validate against API rules
- Create request/response examples
- Suggest status codes and error handling

## When to Use
- Designing new API endpoints
- Updating API documentation
- Planning backend features

# Code Reviewer Agent

A custom agent for reviewing code against project rules.

## Purpose
Analyzes code changes and verifies compliance with RULES.md files.

## Capabilities
- Read and analyze code files
- Check against frontend/backend rules
- Suggest improvements based on naming conventions
- Verify security best practices

## When to Use
- Before creating a pull request
- After making significant changes
- To audit existing code

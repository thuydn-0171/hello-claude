# Hello MC Thuy React App

A simple React application that displays "Hello MC Thuy".

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm start
   ```

The app will open at `http://localhost:3000` and display "Hello MC Thuy".

## Build

To create a production build:
```bash
npm build
```
